HustleMetrics
Track Your Side Hustle ROI - Know Where Your Time is Best Invested

A free, open-source side hustle tracker that helps you understand which income streams are most profitable. Built with vanilla JavaScript and Tailwind CSS - runs entirely in your browser.

 Features
 Track multiple hustles - Freelancing, content creation, e-commerce, etc.

 Log time & earnings - Simple entry logging with notes

 Real-time ROI analysis - See hourly rates for each hustle

 Top performer insights - Automatically identifies your best hustle

 Export reports - Download detailed text reports

 Privacy-first - 100% local storage, no accounts required

 Responsive design - Works on all devices

 Quick Start
Visit the app: Open index.html in your browser

Add your first hustle: Click "+ Add Hustle" button

Log entries: Track hours worked and money earned

Analyze: View real-time stats and hourly rates

 Tech Stack
HTML5, CSS3, Vanilla JavaScript

Tailwind CSS for styling

Local Storage for data persistence

No backend required
